# FreightDoc — why now

Trade-document preparation benefits from a workflow that preserves facts, makes deterministic requirements visible, and routes uncertainty to a person. FreightDoc packages those capabilities in a small web workflow with review history and exportable dossiers.

This is not a claim that a model can make a binding classification or compliance determination. Optional suggestions can accelerate preparation, while local rules, provenance/fallback labels, and human review keep uncertainty explicit.
